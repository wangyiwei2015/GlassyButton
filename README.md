# GlassyButton

### Usage

This will automatically decide to use liquid glass (iOS 26+) or a similar one as replacement.

Available for `Button`, `NavigationLink`, `Link`, `Menu` and other buttons.

```swift
import GlassyButton

Button {
    // Action and/or Label
}.glassyButton()

Button {
    // Action and/or Label
}.glassyButton(.accentColor)

```

### Installation

#### Swift Package Manager

To integrate GlassyButton into your Xcode project using Swift Package Manager, follow these steps:
1. Open your project in Xcode.
2. Go to `File` > `Add Packages...`.
3. In the search bar, enter the URL of the GlassyButton repository: `https://github.com/wangyiwei2015/GlassyButton`.
4. Select the GlassyButton package from the list and click `Add Package`.
5. Choose the target(s) where you want to add the GlassyButton package and click `Add Package`.

#### Manually

Copy file `GlassyButton.swift` and use directly :D
